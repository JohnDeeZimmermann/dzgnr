# Dzgnr Concept

Dzgnr is supposed to be a tool with which AI agents can easily creates designs:
* Business Cards
* Flyers
* Banners
* Posters
* etc.

Basic Idea:
Point to an HTML with dimensions etc. -> Out comes a properly sized CMYK PDF file. 

It does contain a lot of builtin niceties and sane defaults. It also supports any fonts that can be found in Google Fonts. 

The size metric of choic is cm.
